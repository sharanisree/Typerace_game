# Typing Speed Test ⌨️

## 📌 Description
This is a simple Python project that checks typing speed and accuracy.

## 🚀 Features
- Calculates typing speed (Words Per Minute)
- Checks if the sentence is correct
- Allows multiple attempts
- Shows highest typing speed

## 🛠️ Technologies Used
- Python
- time module

## ▶️ How to Run

1. Install Python
2. Download the file
3. Open terminal / command prompt
4. Run this command:

   python typerace.py
